# Task 2 - Credit Risk Prediction

## Objective

The objective of this project is to predict loan approval status using machine learning techniques.

## Dataset

Loan Prediction Dataset from Kaggle.

## Features Used

- Gender
- Married
- Dependents
- Education
- ApplicantIncome
- LoanAmount
- Credit_History
- Property_Area

## Steps Performed

- Data loading
- Missing value handling
- Label encoding
- Data visualization
- Train-test split
- Logistic Regression model training
- Model evaluation

## Visualizations

- Applicant income distribution
- Correlation heatmap
- Confusion matrix

## Model Used

- Logistic Regression

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix

## Outcome

The model was trained successfully to predict loan approval status based on applicant information.
