# Task 3 - Customer Churn Prediction

## Objective

Predict whether a bank customer is likely to leave the bank.

## Dataset

Churn Modelling Dataset from Kaggle.

## Features Used

- Credit Score
- Geography
- Gender
- Age
- Balance
- Number of Products
- Estimated Salary

## Steps Performed

- Data cleaning
- Label encoding
- Data visualization
- Train-test split
- Random Forest model training
- Model evaluation
- Feature importance analysis

## Visualizations

- Churn distribution
- Age vs churn
- Correlation heatmap
- Confusion matrix
- Feature importance

## Model Used

- Random Forest Classifier

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix

## Outcome

The model successfully predicted customer churn and identified important features influencing customer exit behavior.
